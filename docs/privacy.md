# Privacy Policy

Last updated: May 11, 2026

Upgrade Copilot is currently a free Codex plugin distributed through a public custom marketplace source.

## Current Plugin

The current plugin contains Codex skills, documentation, and example configuration. It does not operate a hosted Upgrade Copilot backend and does not collect personal data directly.

When you use the plugin inside Codex, your use remains subject to your Codex, OpenAI, GitHub, and local environment settings.

## Future Premium Services

Future premium features may include hosted repository scans, GitHub App automation, CI analysis, migration intelligence, or license-gated MCP tools. Those services should have their own explicit authentication, data handling, retention, and deletion policies before launch.

## Repository Data

If a future service scans repositories, it should only access repositories you explicitly authorize and should clearly disclose what metadata, code, dependency manifests, logs, or CI results are processed.
